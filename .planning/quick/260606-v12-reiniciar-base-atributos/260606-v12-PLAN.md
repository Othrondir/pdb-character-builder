# Quick Plan: Reiniciar base de atributos

## Objetivo

Corregir el boton `Reiniciar base` del panel de atributos para que restaure solo las caracteristicas base y no borre la raza, subraza, alineamiento ni modificadores raciales ya seleccionados.

## Pasos

1. Reproducir el comportamiento actual con una prueba enfocada del panel de atributos.
2. Anadir una accion parcial al store de fundacion para resetear `baseAttributes`.
3. Cablear `Reiniciar base` a esa accion parcial.
4. Ejecutar Vitest enfocado y typecheck.
