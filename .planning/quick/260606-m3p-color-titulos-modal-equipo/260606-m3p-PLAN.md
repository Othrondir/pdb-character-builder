# Objetivo

Corregir el color de los titulos de los modales `Tipo de Escudo` y
`Tipo de Armadura` para que coincidan con el color del titulo `Hoja de
personaje`.

# Alcance

- Solo CSS del modal de simulacion de equipo.
- No cambiar comportamiento ni calculos de CA.

# Validacion

- Build del planner.
- `git diff --check`.

# Resultado

- Los titulos de los modales de simulacion de equipo usan `var(--color-text)`,
  el mismo color de texto claro que la hoja de personaje.
