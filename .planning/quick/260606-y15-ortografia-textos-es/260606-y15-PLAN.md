# Quick Plan: Ortografia de textos en espanol

## Objetivo

Revisar y corregir faltas ortograficas, acentos y usos de espanol de Espana en textos visibles de la app.

## Alcance

- Incluir textos UI y copy runtime en `apps/planner/src`.
- Incluir capas curadas runtime cuando alimentan textos visibles.
- No editar `compiled-*.ts` generados salvo decision explicita posterior.
- No cambiar textos de tests excepto cuando bloqueen la nueva copy visible.

## Pasos

1. Barrer strings visibles y detectar faltas recurrentes.
2. Corregir copy en ficheros fuente no generados.
3. Actualizar tests afectados por cambios de texto.
4. Ejecutar tests enfocados, typecheck y diff check.
