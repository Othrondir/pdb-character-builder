# Quick Plan: Mobile layout safety

## Objetivo

Evitar que la interfaz rompa o recorte estilos en pantallas pequenas sin alterar el layout de escritorio.

## Alcance

- Cambio CSS-only en `apps/planner/src/styles/app.css`.
- Reglas acotadas a `@media (max-width: 767px)`.
- Sin redisenar componentes ni cambiar JSX.

## Pasos

1. Anadir una capa movil conservadora para que el shell use una columna y scroll seguro.
2. Hacer que las pantallas de seleccion internas pasen a una columna en movil.
3. Permitir que tablas y superficies anchas hagan scroll horizontal local, no de pagina.
4. Ejecutar typecheck, tests focalizados de shell/resumen y `git diff --check`.
