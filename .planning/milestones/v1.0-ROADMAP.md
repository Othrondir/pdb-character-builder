## Overview

Static web character planner for Neverwinter Nights 1 Enhanced Edition tailored to Puerta de Baldur server rules + custom content. Mirrors NWN2DB workflow, NWN1 visual identity, deploys to GitHub Pages, no backend. Spanish-first.

Core value delivered: a player can build a Puerta de Baldur character from level 1 to 20 (range extended from 1-16 in Phase 12.6) with strict server-valid validation and share that exact build reliably via URL or JSON.
