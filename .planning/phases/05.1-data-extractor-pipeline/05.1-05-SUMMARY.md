## Decisions Made
- **NWN color code stripping:** TLK strings contained `<c###>` color tags from the game engine. A regex strip was added to the emitter to clean Spanish labels before catalog emission.
- **Feat column name fix:** cls_feat tables use a `FEAT_` prefix convention for column names. The feat assembler was updated to match this convention instead of raw 2DA column names.
- **Deity as null:** No deity 2DA exists in nwsync or base game. The server manages deity restrictions through scripting and forum rules. Emitted as a null export with documentation comment.

- **Found during:** Post-extraction review (Task 2)

- **Fix:** Added regex stripping of NWN color codes in the TLK resolver before strings reach assemblers
- **Files modified:** packages/data-extractor/src/assemblers/*.ts (string cleanup)
- **Committed in:** 7bcba66

**2. [Rule 1 - Bug] Feat column name mismatch in cls_feat tables**
- **Found during:** Post-extraction review (Task 2)
- **Issue:** cls_feat tables use `FEAT_*` prefixed column names, not the raw column names the assembler was looking for
- **Fix:** Updated feat assembler to use the correct `FEAT_` prefix convention
- **Committed in:** 7bcba66
