# Phase 05.2 Plan 01: Dark NWN1 Theme Foundation Summary

Dark NWN1 replica palette with sharp-cornered double-border panels, gold accent borders, and three reusable UI primitive components (NwnFrame, NwnButton, ActionBar).

## What Was Done

### Task 1: Rewrite CSS tokens, fonts, and full app.css for dark NWN1 theme
- **tokens.css**: Complete rewrite replacing 12 light-theme tokens with 12 dark NWN1 tokens. Removed `--color-parchment`, `--color-ink`, `--color-bronze`, `--shadow-panel`, `--color-panel-glow`, `--color-foundation-canvas`, `--color-foundation-panel`, `--color-selection-fill`. Added `--color-surface`, `--color-panel`, `--color-gold`, `--color-text`, `--color-text-muted`, `--color-selection`, `--color-selection-border`, `--color-blocked`, `--border-frame-inner`, `--border-button`, `--border-button-muted`.
- **fonts.css**: Replaced Cormorant Garamond 700 import with 400 import. Only 400 and 600 weights loaded.
- **app.css**: Full rewrite (930 lines to 530 lines). Body background inverted to `#0a0a0a`. All 17 `border-radius` values set to 0. All ~40 hardcoded `rgba()` values replaced with token references. All `box-shadow` declarations removed. All `font-weight: 700` replaced with 600. Typography density reduced (18px uppercase titles, 14px body). NwnFrame, NwnButton, and ActionBar CSS classes added. Scrollbar styling and `prefers-reduced-motion` media query added.
- **Commit**: `705a7ab`

### Task 2: Create NwnFrame, NwnButton, and ActionBar React components
- **nwn-frame.tsx**: Renders a sharp-cornered double-border NWN1 panel. Accepts `as` (aside/div/nav/section), `children`, and `className` props.
- **nwn-button.tsx**: Renders buttons with primary (gold border), secondary (muted border), auxiliary (muted, 400 weight), and icon variants. Uses `ComponentPropsWithoutRef<'button'>` spread pattern.
- **action-bar.tsx**: Bottom action bar with Aceptar/Cancelar buttons and optional `leftContent` slot. Imports and uses NwnButton internally.
- **Commit**: `c4d8b5a`
