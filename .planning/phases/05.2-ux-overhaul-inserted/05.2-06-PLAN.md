  <name>Task 1: Visual and functional verification of the NWN1 UX overhaul</name>
  <files></files>
  <action>
Start the dev server with `cd apps/planner && npx vite dev` and open http://localhost:5173. Walk through the verification steps below, comparing each screen against the NWN1 reference screenshots in `legacyinterface/`. Present findings to the user for approval.

What was built: Complete NWN1 UX overhaul -- dark theme with golden borders, creation stepper replacing the 7-section sidebar, list+detail center content, tabbed character sheet, GUI-density typography, and all feature boards adapted to the new pattern.

Verification steps for the user:

1. **Overall visual impression (D-09, D-10, D-11):**
   - Compare the planner with `legacyinterface/Captura de pantalla 2026-04-15 204353.png` (the main creation screen reference).
   - Verify: dark background (#0a0a0a), golden borders (#c8a84e), light text (#e8d5a0), NO rounded corners, NO drop shadows, sharp NWN1 frames with double-border effect.

2. **Typography density (D-13, D-14):**
   - Title bars should be UPPERCASE with wide letter-spacing (~18px, not ~36px).
   - Body text should be 14px and dense, matching game text panels.
   - Compare with `legacyinterface/Captura de pantalla 2026-04-15 204607.png` (class selection) for text density reference.

3. **Stepper navigation (D-15, D-16):**
   - Left column should show "Creacion de personajes" heading, then ORIGEN section with 4 step buttons (Raza, Alineamiento, Deidad, Atributos), then PROGRESION section with 4x4 level grid.
   - Click "Raza" -- center content should show race selection list on left, description on right.
   - Select a race. Raza step should show as complete with a checkmark and race name summary.
   - Click "Alineamiento" -- center should show alignment options.
   - Continue through all origin steps.

4. **Character sheet (D-06):**
   - Right column should show "Hoja de personaje" heading with 4 tabs (Estadisticas, Habilidades, Dotes, Conjuros).
   - Stats tab should show 6 ability scores with values after you configure attributes.
   - Derived stats (CA, PG, BAB, saves) should appear.
   - Compare density with `legacyinterface/Captura de pantalla 2026-04-15 204629.png` (completed creation with character sheet).

5. **Level progression (D-08, D-16):**
   - Click a level button in the 4x4 grid.
   - Sub-steps (Clase, Habilidades, Dotes, Conjuros) should appear.
   - Click Clase -- center should show class selection screen.
   - Click Habilidades -- center should show skill distribution.

6. **Responsive (optional):**
   - Resize below 1024px -- stepper should collapse.
   - Resize below 768px -- single column.

7. **Functional check:**
   - Complete a full origin (race + alignment + deity + attributes).
   - Add a class to level 1.
   - Allocate skills for level 1.
   - Verify no console errors and no blank screens.
  </action>
