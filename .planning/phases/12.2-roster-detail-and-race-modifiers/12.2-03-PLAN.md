NWN alignment hex-mask convention (from NWN 2DA docs + Puerta AlignRestrict usage):
- Each bit flags one of 9 alignments. Bit indices per NWN2 / NWN Baldur's Gate tradition:
  - 0x01 = LG   (bit 0)
  - 0x02 = LN   (bit 1)
  - 0x04 = LE   (bit 2)
  - 0x08 = NG   (bit 3)
  - 0x10 = TN   (bit 4)
  - 0x20 = NE   (bit 5)
  - 0x40 = CG   (bit 6)
  - 0x80 = CN   (bit 7)
  - 0x100 = CE  (bit 8)
- `0x00` = no restriction (all 9 alignments allowed).
- `InvertRestrict == "1"` — the mask lists FORBIDDEN alignments; the allowed set is the complement over the 9-alignment universe.
- `InvertRestrict == "0"` — the mask lists ALLOWED alignments.
- SPECIAL CASE: Some PDB rows (e.g. druid row 3: `AlignRestrict: "0x01"`, `InvertRestrict: "1"`) use `AlignRstrctType` as a secondary layer (law/chaos axis vs good/evil axis). For the scope of this plan, decode ONLY `AlignRestrict` + `InvertRestrict`; treat `AlignRstrctType` as informational. If AlignRestrict is `0x00` (no mask), emit `undefined` (no gate). If InvertRestrict=1 and mask=0x00 → no gate either.

    /**
     * Phase 12.2-03 (D-04) — decode the NWN hex alignment mask from a
     * CompiledClass.prerequisiteColumns.AlignRestrict entry. Returns the
     * allowed CanonicalId alignment list, or undefined when no gate applies.
     *
     * Hex-bit convention (NWN Aurora 2DA):
     *   0x01 = LG, 0x02 = LN, 0x04 = LE,
     *   0x08 = NG, 0x10 = TN, 0x20 = NE,
     *   0x40 = CG, 0x80 = CN, 0x100 = CE.
     *
     * InvertRestrict="1" inverts the mask: the listed alignments are
     * FORBIDDEN; the allowed set is the complement. `0x00` inverted is still
     * `0x00` — no gate either way.
     */
    const ALIGNMENT_BITS: ReadonlyArray<readonly [number, CanonicalId]> = [
      [0x001, 'alignment:lawful-good' as CanonicalId],
      [0x002, 'alignment:lawful-neutral' as CanonicalId],
      [0x004, 'alignment:lawful-evil' as CanonicalId],
      [0x008, 'alignment:neutral-good' as CanonicalId],
      [0x010, 'alignment:true-neutral' as CanonicalId],
      [0x020, 'alignment:neutral-evil' as CanonicalId],
      [0x040, 'alignment:chaotic-good' as CanonicalId],
      [0x080, 'alignment:chaotic-neutral' as CanonicalId],
      [0x100, 'alignment:chaotic-evil' as CanonicalId],
    ];

    export function decodeAlignRestrict(
      mask: string | null,
      inverted: string | null,
    ): CanonicalId[] | undefined {
      if (!mask) return undefined;
      const parsed = parseInt(mask, 16);
